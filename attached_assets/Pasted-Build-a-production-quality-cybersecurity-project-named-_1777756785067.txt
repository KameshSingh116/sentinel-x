## ⚙️ CORE FEATURES

### 1. System Monitoring

* Use `psutil`
* Track CPU, memory, processes
* Detect:

  * CPU > 80%
  * Unknown processes

---

### 2. Network Monitoring

* Track active connections
* Extract IP addresses
* Detect suspicious/repeated connections

---

### 3. Anomaly Detection

* Rule-based detection:

  * Sudden spikes
  * Repeated patterns
* Design code to allow ML upgrade later

---

### 4. REST API (MANDATORY)

Create endpoints:

* `/logs`
* `/alerts`
* `/status`
* `/trigger-test-alert`

Return JSON responses.

---

### 5. Alert System

* Use `smtplib`
* Send email alerts
* Store alerts in database

---

### 6. Logging System

* JSON + SQLite
* Fields:

  * timestamp
  * event
  * severity
  * description
